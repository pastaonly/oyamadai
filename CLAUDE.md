# プロジェクト概要

本プロジェクトでは、**ハッピー探偵団**について検討する。

ハッピー探偵団は、ハッピーロード尾山台商店街が運営するウェブメディア（<https://happy.oyamadai.com/>）。尾山台に愛着を持つ仲間たちが、地域住民の「やりたいこと」の実現を応援し、尾山台をより身近に感じてもらうことを目的としている。

本リポジトリでは、広報体制・運用方針、課題管理、関連ドキュメントの整理を行う。

# ディレクトリ構成

```
oyamadai/
├── CLAUDE.md                    # 本ファイル（プロジェクト概要・構成）
└── happy-tantei-dan/            # ハッピー探偵団に関する検討資料
    ├── management.md            # 課題一覧・マイルストーン・活動記録
    ├── policy.md                # 体制・方針メモ
    ├── document/                # 関連ドキュメント
    └── activity-log-photos/     # 活動記録の写真
```
