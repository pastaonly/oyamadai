# ハッピー探偵団

ハッピーロード尾山台商店街が運営するウェブメディア「**ハッピー探偵団**」の検討・運用リポジトリ。
尾山台に愛着を持つ仲間たちが、地域住民の「やりたいこと」の実現を応援し、尾山台をより身近に感じてもらうことを目的としている。

- 公開サイト: <https://happy.oyamadai.com/>

## 主要リンク

| 種別 | 内容 | リンク |
| --- | --- | --- |
| 📊 管理シート（正本） | 受付管理・制作管理・会員店舗リスト・予算メモ。**運用の最新版はこちら** | [Googleスプレッドシート](https://docs.google.com/spreadsheets/d/1iDDUZNVUm1FfA0x_cwvwQoELgDVs3Xd-RSeaHR2ERFY/edit) |
| 📝 議事録 | ハッピーロードメディア事業 議事録統合サマリー（決定事項・タスク・予算・運用フロー） | [Googleドキュメント](https://docs.google.com/document/d/133JFhN_sZK8h_sOkcguRFe6V3qJ7k844hN4tvy3KFoY/edit) |
| 🌐 公開サイト | ハッピー探偵団（ウェブメディア本体） | <https://happy.oyamadai.com/> |

> **正本について**: 受付・制作・店舗・予算の最新状況は上記 Googleスプレッドシートを正とする。
> 本リポジトリの `document/ハッピー探偵団_マネジメント.xlsx` は、フォーマットの原型／スナップショットとして保持する（助成金報告の証跡用）。
