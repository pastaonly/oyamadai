## 論点の叩き台

### 1. 本メディアの可能性と位置付け

検討単位は「記事」ではなく **メディア全体（Web記事＋Instagram＋瓦版）**。鍵は「誰のための媒体か」を二層で捉えること。

| 二面性 | 価値 | お金の出どころ |
| --- | --- | --- |
| **読者＝住民向け**（公共財） | 認知・愛着・コミュニティ・担い手育成 | 商店街予算で支える |
| **顧客＝店舗向け**（販促媒体） | 送客・PR・MEO/Instagram連動 | 店舗の協賛・掲載・タイアップ＝**自主財源** |

- **両面市場**：住民の読者基盤（Instagram 5,000）を育てるからこそ、店舗にとっての広告価値が出る。
- → **フォロワー目標は虚栄指標ではなく"財源の源泉"**。SNS数値目標と自走財源は一本の線。
- 主軸（＝何で稼ぐか）の方向感：**「住民向けで基盤を作り、店舗向けで稼ぐ」二層構造**を推し。

### 2. 適切な運営母体

決定的な整理は **「お金の器」と「編集・運用の主体」を分ける**こと。

```
【お金の器・契約・雇用】  おやまだい株式会社（おや株）
    ├─ 記事事業（紙芝居）
    ├─ Instagram事業 ← 運用もおや株で受ける形に集約
    └─ 収入：商店街予算（基礎）＋店舗協賛/タイアップ（自主）

【編集・運用の主体】
    ├─ 記事：学生バイト編集部 ＋ シップヤード/黒川さん監修
    └─ Instagram：松井さん ＋ 運用サポーター(店舗)

【公共性・予算承認】  商店街（理事会）← 黒川さん(理事)が橋渡し

【担い手レイヤー】  学生バイト ＋ 運用サポーター(店舗)
   ※「記事のバイト化」と「SNS運用サポーター育成」は同じ一手
```

- **器＝おや株に集約**：法人格・調査広報事業の実績（第2期 売上184万）あり。記事もInstagramも対外契約も一体会計で受けられる。
- **黒川さんの三役を活かす**：制作（実現性）／店舗オーナー（協賛が成立するかの生の検証）／理事（最低ライン予算を理事会に橋渡し）。
- **クリーンにしておく設計**：おや株＝器、シップヤード＝受注者、黒川さん＝理事として予算を通す側。**お金の流れ（誰が誰に払うか）を透明に**しておくと、商店街予算で支える局面で説明しやすい。

### 3. 複数収入源の可能性

コストが下がる前提（後述）なら、「大口1本」より **「薄く複数」** で黒字化が射程。

| 収入源 | 規模感 | 位置づけ |
| --- | --- | --- |
| 商店街予算 | 基礎（薄く確実） | 公共財部分の土台。最低ライン確保が第一目標 |
| 店舗協賛・掲載料 | 月数千円×複数 / 特集スポンサー | **本命の自主財源** |
| Instagram運用代行・受託 | 案件次第 | 既存スキルの収益版（運用サポーター事業の有料化） |
| タイアップ記事（不動産・大学・行政） | 中〜大口 | 媒体力が前提 |
| グッズ（オッポン等） | 小 | 収益より愛着・話題づくり |
| イベント連動 | 小〜中 | おや株イベント事業（78万）と相互送客 |

方針案：**基礎＝商店街予算 ＋ 伸びしろ＝店舗協賛/タイアップ**。

### 4. 制作モデル（コスト構造の転換）

- **記事**：外注ベース 〜60万 → **学生バイトベースで〜10万に圧縮**できる想定（要実証）。
- **Instagram**：広告運用費は**現金コストで圧縮しにくい**。設計は「**今年＝補助金で広告を打って5,000まで伸ばす → 補助金後＝維持モード＋運用サポーター(店舗)で自走**」。
- **隠れコスト＝編集長機能**：品質・進行・公開判断を誰がどれだけの工数で持つか。バイトは手を増やすが責任は別途必要。→ 第2回で具体化。

## 今年度（2026）やるべきこと

合言葉：**「補助金があるうちに、補助金なしで回る形をテストする」**

- **制作モデルの実証**：今年度の記事1本を学生バイトベースで試作し、実コスト・品質・編集工数を実測（=10万仮説の検証）
- **編集長機能の試運用**：管理システム（受付/制作シート）を回しながら監修・進行の担い手を見極める
- **収益の小実験**：店舗協賛 or タイアップを1件でも有償で。オッポン小ロットも
- **読者基盤の積み上げ**：補助金予算が使える今のうちに広告でInstagramの「山」を作る（5,000目標）
- **意思決定と予算化**：第2〜3回で母体・基礎財源を決め、**商店街の来年度予算サイクルに間に合わせる**
