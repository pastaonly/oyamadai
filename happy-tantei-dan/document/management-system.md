# ハッピー探偵団 マネジメントシステム（簡易版・構成案）

応募フォームや口頭で届く「やりたいこと」の依頼を、受付から記事公開・プロモーションまで一気通貫で管理するための仕組み。
**1つのスプレッドシート**内に、以下のシートを持つ構成とする。

| シート | 持ち方 | 役割 |
| --- | --- | --- |
| `受付管理` | 横持ち（1行＝1件） | 届いた依頼の一覧。対応方針・ステータスを管理し、企画化したら制作シートへ繋ぐ |
| `制作_◯◯`（企画ごと） | 縦持ち（1行＝1項目） | 企画化した案件の制作業務を管理。企画が増えるたびにシートを追加 |
| `会員店舗` | 横持ち（1行＝1店舗） | 商店街会員店舗のリスト。取材先・つなぎ先の候補として参照 |
| `マスタ`（任意） | ― | プルダウンの選択肢一覧（方針・ステータス・担当など）をまとめる |

---

## 1. 受付管理シート（`受付管理`）

届いた依頼を1行1件で管理。先に検討したフォーム項目をそのまま取り込み、その右側に「探偵団としてどう対応するか」の管理項目を足す。

### 列構成

| 列 | 項目 | 内容・選択肢 |
| --- | --- | --- |
| A | 受付ID | 例：`R-001`（連番） |
| B | 受付日 | 届いた日付 |
| C | 受付経路 | フォーム／口頭／メール／SNS／その他 |
| D | おなまえ | ニックネーム可 |
| E | メールアドレス | |
| F | 年代 | 10代以下／10代／20代／30代／40代／50代／60代／70代以上 |
| G | 尾山台との関わり | 住んでいる／近隣に住んでいる／通勤・通学／お店・活動／ときどき訪れる／その他 |
| H | やりたいこと | 依頼内容（本文） |
| I | 取材・掲載連絡OK | ◯／―（フォームのチェック） |
| J | 対応方針 | ↓下記参照 |
| K | 対応ステータス | ↓下記参照 |
| L | 企画化リンク | 企画化した場合の制作シート名（例：`制作_結婚式`） |
| M | 担当 | 受付対応の担当者 |
| N | 最終更新日 | |
| O | メモ | 検討経緯・本人とのやりとりなど |

### J列：対応方針（選択肢）

「叶える窓口ではなく、取材・つなぎのきっかけとして扱う」という方針を反映。

- `取材して企画化する` … 制作管理シートを起こす
- `町の人・お店につなぐ` … 紹介のみ（記事化はしない）
- `情報提供・アドバイス` … 探偵団から情報を返すのみ
- `保留` … 時期・条件を見て再検討
- `見送り（お礼返信）` … 対応はしないが感謝を返す

> **【補助金要件による方針メモ】**
> 東京都補助金の要件により、企画は「○○がやりたい」型から「**○○で困っている**」型・**商店街の活動PR型**へシフトする方針（議事録決定）。
> 受付した依頼を企画化するかの判断時は、「商店街の活動・困りごと解決が見えるコンテンツになるか」を観点に加える。
> フォームの見せ方（前提文）も、この方針と整合させて要調整。

### K列：対応ステータス（選択肢）

- `新規受付`
- `内容確認中`
- `方針検討中`
- `本人へ連絡済み`
- `企画化（→制作へ）`
- `クローズ`

## 2. 制作管理シート（`制作_◯◯` / 企画ごと）

企画化が決まったら、この縦持ちシートを1企画＝1シートで複製して使う。
**縦に業務項目が並び、各項目に説明文・担当・期日・進捗・メモを記入**していく形。

### 列構成（共通ヘッダー）

| 列 | 項目 |
| --- | --- |
| A | 区分 |
| B | 項目 |
| C | 説明（この項目で何をするか） |
| D | 担当 |
| E | 期日 |
| F | 進捗 |
| G | メモ |

### F列：進捗（選択肢）

`未着手` ／ `対応中` ／ `完了` ／ `保留` ／ `不要`

## 運用メモ

- **受付 → 制作の流れ**：受付管理で `対応方針＝取材して企画化する` になったら、制作シートを1枚複製。`L列：企画化リンク` にシート名を書いて相互に辿れるようにする。
- **シートの増やし方**：制作シートはテンプレート（`制作_テンプレート`）を1枚用意し、企画ごとに複製してリネーム。
- **第4弾「まどかさん」**：2026/6/14 取材予定（management.md 参照）。最初の1企画として、このテンプレートで試運用するとよい。
- **助成金報告との連携**：制作シートの撮影日・公開日が、年度末報告の活動記録の素材になる。
